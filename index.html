Core Courses

AWD1000 - Web Development Technologies

Students will get hands-on experience in the latest HTML, CSS, and JavaScript technologies.

AWD1100 - Programming Fundamentals w/ C#

This course offers an intro to C# programming, e.g., object-oriented and relational databases.

AWD1111 - Database Driven Web Development

This course covers server-side web apps utilizing the MERN stack (MongoDB, Express, React, NodeJS).

AWD1115 - Database Driven Web Development II

Students will implement Model View Controller (MVC) and relational databases to create C# Web apps.
